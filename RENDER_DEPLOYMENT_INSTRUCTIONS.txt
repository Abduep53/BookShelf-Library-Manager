📋 STEP-BY-STEP DEPLOYMENT (5 MINUTES)
═══════════════════════════════════════════════════════════════════

STEP 1: Open Render Dashboard
-------------------------------
URL: https://dashboard.render.com/

Action:
- Sign in to your Render account (or create free account)
- Keep this window open


STEP 2: Create Blueprint Deployment
------------------------------------
In Render Dashboard:

1. Click "New +" button (top right)
2. Select "Blueprint"
3. Connect GitHub account (if not connected)
   - Click "Connect GitHub"
   - Authorize Render
4. Find your repository:
   - Search: "BookShelf-Library-Manager"
   - Select the repository
5. Click "Connect"


STEP 3: Apply Blueprint
------------------------
Render will detect render.yaml automatically and show:

Services to create:
  ✓ bookshelf-app (Web Service - Python)
  ✓ bookshelf-db (PostgreSQL Database)

Environment Variables (auto-configured):
  ✓ SECRET_KEY (auto-generated)
  ✓ DEBUG=False
  ✓ DATABASE_URL (from database)
  ✓ PYTHON_VERSION=3.12.0

Action:
- Review the configuration
- Click "Apply" button
- Wait for services to be created (30 seconds)


STEP 4: Add OpenAI API Key
---------------------------
IMPORTANT: Add your OpenAI API key manually

1. Go to "bookshelf-app" service (click on it)
2. Navigate to "Environment" tab (left sidebar)
3. Click "Add Environment Variable"
4. Enter:
   Key: OPENAI_API_KEY
   Value: [paste your OpenAI API key]
5. Click "Save Changes"

⚠️  This will trigger an automatic redeploy


STEP 5: Monitor Deployment
---------------------------
Build process started automatically!

Expected time: 5-10 minutes

Build steps:
1. Installing dependencies from requirements.txt
2. Collecting static files
3. Running database migrations
4. Starting Gunicorn server

Monitor progress:
- Go to "bookshelf-app" service
- Click "Logs" tab
- Watch build progress in real-time

Wait for status to show: "Live" (green indicator)

═══════════════════════════════════════════════════════════════════
🛠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Build Fails:
- Check build logs in Render dashboard
- Verify all dependencies in requirements.txt
- Ensure build.sh has correct permissions

App Won't Start:
- Verify OPENAI_API_KEY is set correctly
- Check start command: gunicorn BookShelf.wsgi:application
- Review application logs

Database Issues:
- Ensure DATABASE_URL is connected
- Run migrations: Dashboard → Shell → python manage.py migrate

AI Features Not Working:
- Verify OPENAI_API_KEY is correct
- Check OpenAI API quota/billing
- Review application logs for errors
