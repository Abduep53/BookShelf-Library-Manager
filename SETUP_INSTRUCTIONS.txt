    QUICK SETUP (3 STEPS)
═══════════════════════════════════════════════════════════════

STEP 1: Install Dependencies
────────────────────────────────────────────────────────────────
Run this command in your project directory:

    pip install -r requirements.txt

This installs:
  • openai>=1.0.0
  • python-dotenv>=1.0.0

────────────────────────────────────────────────────────────────
STEP 2: Create .env File with Your OpenAI API Key
────────────────────────────────────────────────────────────────

A. Get your OpenAI API key:
   1. Visit: https://platform.openai.com/api-keys
   2. Click "Create new secret key"
   3. Copy the key (starts with "sk-...")

B. Create .env file in project root:
   
   Windows PowerShell:
   New-Item -Path .env -ItemType File
   notepad .env

   macOS/Linux:
   touch .env
   nano .env

C. Add this line to .env (replace with your actual key):
   
   OPENAI_API_KEY=sk-your-actual-api-key-here

   IMPORTANT: Replace "sk-your-actual-api-key-here" with your real key!

D. Save and close the file

────────────────────────────────────────────────────────────────
STEP 3: Restart Django Server
────────────────────────────────────────────────────────────────

Stop the server (Ctrl+C) and restart:

    python manage.py runserver

    TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Error: "OpenAI API key not configured"
→ Check that .env file exists in project root
→ Verify .env contains: OPENAI_API_KEY=sk-...
→ Restart Django server

Error: "Invalid API key"
→ Verify you copied the entire key (starts with sk-)
→ Check for extra spaces or quotes
→ Generate a new key if needed

AI not responding / taking too long
→ Check your internet connection
→ Verify OpenAI API status: https://status.openai.com
→ Try again (occasional timeouts are normal)
